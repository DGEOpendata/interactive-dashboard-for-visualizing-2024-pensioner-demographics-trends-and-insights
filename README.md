# Interactive Dashboard for Visualizing Pensioner Demographics

This repository provides the source code for creating an interactive dashboard to visualize and analyze the Pensioners Distribution for 2024 dataset. The dashboard is built using Python, Dash, and Plotly, providing an intuitive and dynamic interface for stakeholders to explore demographic trends among pensioners.

## Features
- Interactive pie charts and bar graphs for visualizing gender and overall pensioner distributions.
- Dropdown menu for selecting specific quarters of 2024.
- Downloadable visualizations and filtered data for offline analysis.
- Integration with related documentation for additional context.

## Prerequisites
To run this project, you will need:
- Python 3.7+
- pip (Python package manager)

## Installation
1. Clone this repository:
   bash
   git clone <repository-url>
   cd <repository-name>
   
2. Install the required Python packages:
   bash
   pip install -r requirements.txt
   

## Dataset
Download the following files and place them in the root directory of this project:
1. [Distribution of Pensioners per Gender.xlsx](link-to-dataset)
2. [Distribution of Pensioners.xlsx](link-to-dataset)

## Running the Dashboard
To start the dashboard, execute the following command:
bash
python app.py

Open your web browser and navigate to `http://127.0.0.1:8050/` to access the dashboard.

## Usage
1. Select the desired quarter from the dropdown menu to update the visualizations.
2. View the gender distribution and overall distribution of pensioners for the selected quarter.
3. Download visualizations or filtered data using the export options.
